  1. In Slack, go to More (•••) → Tools → Workflows
  2. Click +, name it (e.g. Time Tracker), click Next
  3. Choose Starts with a webhook, click Continue
  4. Click Add variable: name it message, type Text, click Save
  5. Copy the Webhook URL and paste it in the field above
  6. Click Add Step, choose Send a message to a channel
  7. Choose the channel to post to
  8. In the message body, click Insert a variable
  9. Select message from the list, then click Save
  10. Click Publish — your workflow is live
